# AWS EC2 Website Hosting Project 🚀

## Project Overview
This project demonstrates deployment of a live website on AWS EC2 using Ubuntu Linux and Nginx web server.

## Features
✅ Live website hosting  
✅ Secure SSH access  
✅ Public IP based deployment  
✅ HTTP traffic configuration  
✅ Linux server management  

---

## Steps Performed

### 1. Created IAM Admin User
Configured secure IAM user for AWS access.

### 2. Launched EC2 Instance
Created Ubuntu EC2 server on AWS cloud.

### 3. Configured Security Groups
Allowed:
- SSH (Port 22)
- HTTP (Port 80)

### 4. Connected Using SSH
Used `.pem` key for secure remote login.

### 5. Installed Nginx
Configured Nginx web server on Ubuntu.

### 6. Hosted Custom Website
Deployed custom HTML webpage successfully.
